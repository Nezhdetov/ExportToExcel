# ExportToExcel
Simple java project for exporting text files to excel using Apache POI  :simple_smile:

## Apache POI runtime dependencies:
```
<dependency>
    <groupId>org.apache.poi</groupId>
    <artifactId>poi</artifactId>
    <version>3.9</version>
</dependency>
```
